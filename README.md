# pi-hole-dns-lists
DNS Lists to use with Pi-Hole DNS Blocker

 - Login to your Pi-Hole
 - Go to Settings
 - Open Blocklists Tab
 
You may remove all default blocklists as they are included again in this list.

List of approved Blocked Domains:
